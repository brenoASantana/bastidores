# Changelog - BACKROOMS\n\nTodas as mudanças notáveis neste projeto serão documentadas neste arquivo.\n\nO formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),\ne este projeto adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).\n\n## [0.1.0] - 2026-04-01\n\n### Added (Adicionado)\n\n#### Core Gameplay\n- ✅ First-person exploration com controles FPS (WASD + Mouse)\n- ✅ Sistema dinâmico de ansiedade (0-100) reativo à zona do mapa\n- ✅ Coleta de 3 objetivos para vitória\n- ✅ Condição de derrota: colapso mental (ansiedade 95+ por 5s)\n- ✅ Objetivo de saída após coleta completa\n\n#### Horror Systems\n- ✅ `HorrorSystem` com cálculo de ansiedade por zona\n  - Zona segura: -0.15/s\n  - Zona normal: +0.5/s\n  - Zona aberta: +0.75/s\n- ✅ Eventos de horror aleatórios (passos distantes, sussurros)\n- ✅ Efeitos visuais por limiares de ansiedade\n  - Vinheta preta\n  - Granulado\n  - Blur\n  - Aberração cromática\n  - Zoom leve\n\n#### Audio & Music\n- ✅ `AudioSystem` com Howler.js\n- ✅ Trilha em 2 camadas (ambient + tension)\n- ✅ Mixer dinâmico reativo à ansiedade\n- ✅ SFX posicionais (placeholder com data URIs)\n- ✅ Lazy loading de Howler para evitar SSR issues\n\n#### 3D Rendering\n- ✅ React Three Fiber com Three.js\n- ✅ Mapa com corredores, paredes, piso, teto\n- ✅ Iluminação fluorescente instável\n- ✅ Colisão AABB simples\n- ✅ Neblina para efeito de profundidade\n- ✅ Marcadores de objetivos (esferas amarelas com luz)\n\n#### State Management\n- ✅ Zustand store com `useGameStore`\n- ✅ Rastreamento de estado do jogo (boot → menu → playing → failed/completed)\n- ✅ Posição e rotação do player em tempo real\n- ✅ Progressão de objetivos\n- ✅ Temporizador de sessão\n\n#### UI & UX\n- ✅ Menu inicial com instruções\n- ✅ HUD in-game com:\n  - Barra de ansiedade\n  - Contador de objetivos\n  - Instruções de controle\n  - Indicador de distorção por ansiedade\n- ✅ Tela de fim (vitória com tempo / derrota com ansiedade final)\n- ✅ Responsive design (desktop + mobile basic)\n- ✅ Pointer lock para câmera imersiva\n\n#### Development & Tooling\n- ✅ Next.js 14 com TypeScript\n- ✅ Tailwind CSS para styling\n- ✅ ESLint & Prettier\n- ✅ Makefile com comandos úteis\n- ✅ `next.config.mjs` otimizado\n- ✅ `.npmrc` com legacy-peer-deps para compatibilidade\n\n#### Documentation\n- ✅ README.md com overview\n- ✅ ARCHITECTURE.md com design técnico\n- ✅ GAMEPLAY.md com guia do jogo\n- ✅ DEVELOPMENT.md com guia dev\n- ✅ CONTRIBUTING.md com diretrizes\n- ✅ QUICK_REFERENCE.md com snippets\n- ✅ CHANGELOG.md (este arquivo)\n\n### Changed (Alterado)\n\n- Downgrade de Next.js 15 → 14.1 (estabilidade)\n- Downgrade de React 19 → 18.2 (compatibilidade com R3F 8)\n- Removed `next.config.ts` → `next.config.mjs`\n- Refactored audioSystem para lazy-load Howler\n- Removido dynamic import de GameContainer (causava erros React)\n\n### Fixed (Corrigido)\n\n- ❌ \"Fog is not exported from @react-three/drei\" → Usado
